# CRM + Chat (в разработке)

## О проекте
Это мой pet-проект — backend для CRM-системы с задачами и чатом.

Я делаю его, чтобы прокачать навыки backend-разработки и собрать нормальный проект, который можно показать работодателю.

---

## Что уже сделано
На данный момент есть базовая логика CRM:

- реализована модель задач (Task)
- сделан полный CRUD через Django REST Framework
- подключена JWT-аутентификация
- настроен Swagger для тестирования API
- добавлены роли пользователей (admin, operator, employee, client)
- реализованы permissions (кто может создавать/редактировать/удалять)
- задачи автоматически привязываются к пользователю (created_by)
- пользователь видит только свои или назначенные задачи

## Как работает API
Основные endpoints:

- GET /api/tasks/ — список задач
- POST /api/tasks/ — создать задачу
- GET /api/tasks/{id}/ — получить задачу
- PATCH /api/tasks/{id}/ — обновить
- DELETE /api/tasks/{id}/ — удалить

Для доступа нужен JWT токен:
Authorization: Bearer <token>
